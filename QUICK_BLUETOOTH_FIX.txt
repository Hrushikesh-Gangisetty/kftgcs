# QUICK REFERENCE: Bluetooth Connection Fix

## Problem: "connection failed: k3 error"
- Occurs in **release builds** only (not in debug)
- Bluetooth connects but navigation still happens
- Indicates ProGuard obfuscation issue

## What Was Fixed
✅ Disabled Log stripping (was hiding real errors)
✅ Protected Okio buffering classes (BufferedSource/Sink)
✅ Protected Kotlin reflection (needed by MAVLink)
✅ Expanded telemetry package protection

## If Still Not Working

**Check these logs:**
```bash
adb logcat | grep -E "(Bluetooth|MAVLink|SharedVM|Connection)"
```

**Common issues:**
1. Device not powered on
2. Device not in pairing mode
3. Permissions not granted
4. Wrong UUID for device module
